# GigShield AI

## Problem
Delivery workers lose income due to external disruptions like heavy rain, floods, or pollution.

## Solution
GigShield AI is an AI-powered parametric insurance platform that protects delivery workers from income loss.

## Persona
Food Delivery Workers (Swiggy/Zomato)

## Disruption Triggers
1. Heavy Rain
2. Flood
3. High Pollution (AQI)

## Platform
Web application.

## Tech Stack
Frontend: React
Backend: Node.js
Database: MongoDB
AI: Python

## Features
- Weekly insurance pricing
- AI risk prediction
- Automatic claim trigger
- Instant payout simulation
